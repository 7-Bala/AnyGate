# Any Gate

Accessibility-first wayfinding assistant for FIFA World Cup 2026.

> Work in progress — full documentation (approach, architecture, assumptions) lands in a later milestone.

## Dev

```bash
npm install
cp .env.example .env   # fill in API keys locally
npm run dev             # frontend (Vite)
npm run server:dev      # backend (Express)
npm test                # unit tests
```
